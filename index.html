A Mágica do Airflow

O Apache Airflow não altera os dados diretamente, ele gerencia quem altera. Ele garante que as rotinas Python (Workers) da Fase 2 rodem no horário certo. Se o sistema do Ministério da Saúde falhar na Fase 2, o Airflow segura a Fase 3 e tenta de novo, enviando um alerta caso não consiga.

Stack 100% Código Aberto

Todo este ecossistema pode ser "conteinerizado" via Docker. Um container roda o PostgreSQL, outro roda os serviços do Airflow, e as rotinas (DAGs) são escritas de forma nativa em Python, garantindo altíssimo controle, flexibilidade e baixo custo a longo prazo para a gestão municipal.

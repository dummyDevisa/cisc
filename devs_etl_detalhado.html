Detalhamento Técnico do Processo

📄 1. Ingestão do Caos

O processo começa onde a saúde pública realmente acontece: no papel. Fichas de notificação e planilhas avulsas são coletadas. O desafio aqui é a variabilidade de formatos e a qualidade da caligrafia.

🤖 2. Visão Computacional

As imagens são processadas por redes neurais que analisam cada campo individualmente. Em vez de uma nota global, a IA atribui um Índice de Confiança por célula, isolando incertezas.

👀 3. Revisão Cirúrgica (HITL)

O sistema destaca apenas os campos duvidosos para o revisor humano. A validação é cirúrgica, reduzindo drasticamente o tempo de processamento manual.

⚙️ 4. Refinaria de Dados

Com os dados digitais, o pipeline executa deduplicação e validação cruzada com bases oficiais (Receita/Cartão SUS) usando scripts Python de alto desempenho.

📍 5. Normalização Geográfica

Nomes de ruas e bairros são corrigidos para padrões oficiais, permitindo o cruzamento exato com mapas de risco climático e manchas de inundação.

🐘 6. Carga no DW/Datalake

O dado higienizado é injetado no coração do servidor, pronto para consumo pela Sala de Situação e cruzamento com fontes climáticas agnósticas.

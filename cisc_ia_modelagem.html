1. Portabilidade: Independência de Modelo

O "Segredo" da Longevidade: Pesos de modelos de IA são voláteis, mas o Dataset é eterno. Se decidirmos trocar a arquitetura da IA, o Universal Training Bucket é usado para transferir todo o conhecimento acumulado pelos revisores humanos para o novo modelo em poucas horas.

2. Estratégia Híbrida e Resiliência

Cloud API Fallback

Caso o volume supere a capacidade local ou o hardware falhe, o sistema roteia automaticamente para APIs externas. A inteligência do CISC não para.

Treino Híbrido

O Training Bucket captura dados de ambos os fluxos, garantindo que o modelo local continue aprendendo mesmo durante o uso temporário da nuvem.

3. Requisitos de Hardware

📟 Recomendado (2026)

  • GPU: NVIDIA RTX 4070 Ti Super (16GB VRAM)
  • VRAM: Uso de bitsandbytes (NF4) para otimização.
  • RAM: 32GB DDR5
  • Disk: NVMe Gen4 (Leitura rápida do Bucket).

Dica de Implementação

"Utilizaremos o modelo para gerar Dados Sintéticos baseados nos templates das fichas da SESMA, acelerando o fine-tuning antes do primeiro grande lote de HITL."
